Based on Deep learning video data the detection of Autism Spectrum Disorder(ASD) is done by making use of RNNs, CNNs and LSTM networks. Made use of videos and then created csv file of all common symptoms manually binary class is used. Traditionally, the diagnosis obtained from clinical assessment is subjective and contains lots of errors. The lack of advanced technique and delays with regard to the diagnosis remain an issue. So the system is to be develop that relies on video inputs for behavior traits in children suffering from ASD which mainly consisting of social interactions, actions, hand movement and repetitive actions.
